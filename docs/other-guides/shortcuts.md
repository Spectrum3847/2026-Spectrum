# Development Environment Shortcuts

Efficient use of your Integrated Development Environment (IDE) and version control tools can significantly speed up your workflow. This guide provides a collection of useful shortcuts for VS Code and other development tasks.

## VS Code Shortcuts

*   **Tab**: Accept an inline suggestion (e.g., from GitHub Copilot).
*   **`Ctrl + Right Arrow`**: Accept the next word of a suggestion.
*   **`F2`**: Refactor Rename – renames all instances of a variable, method, or class throughout your codebase.
*   **`Ctrl + /`**: Toggle line comment for the currently highlighted text.
*   **`Ctrl + Click`** or **`F12`**: Go to Definition – navigate directly to the definition of a symbol (variable, method, class).
*   **`Alt + Shift + F12`**: Find All References – find all places where a symbol is used.
*   **`Ctrl + p`**: Command Bar – quickly search for and open files, run commands, or navigate to symbols.
*   **<kbd>Ctrl</kbd> + <kbd>`</kbd>**: Toggle terminal visibility (show/hide the integrated terminal).
*   **`Ctrl + Shift + V`**: Preview Markdown – useful for seeing how your `README.md` or other Markdown files will render.
*   **`Ctrl + Alt + Left Arrow`**: Split editor screen (e.g., to view two files side-by-side).
*   **`Shift + Alt + Down`**: Duplicate a line or selected block of code.

## Cursor Shortcuts (Multi-Cursor and Selection)

These shortcuts allow for powerful multi-cursor editing and precise text selection.

*   **`Ctrl + Alt + Up/Down`**: Add a cursor above or below the current cursor position.
*   **`Alt + Left Click`**: Place cursors at specific, non-contiguous locations.
*   **`Middle Click + Drag`**: Place cursors on each line or highlight sections of lines as you drag.
*   **`Ctrl + D`**: Select the next occurrence of the currently selected text.
*   **`Ctrl + U`**: Deselect the last selected occurrence (undo `Ctrl + D`).
*   **`Ctrl + Shift + L`**: Select all instances of the currently selected text.
*   **`Alt + Shift + Arrow Keys`**: Expand or shrink the selection (e.g., select current word, then current line, then current block). This is a quick way to edit multiple lines or change many things (e.g., from `public` to `private`).

## Build, Simulate, and Deploy Shortcuts

*   **`Shift + F5`**: Deploy code to the robot.
*   **`F5`**: Start simulation/debugging.
*   **`F4`**: (Can be remapped) Build the project.

## Working with GitHub in VS Code

VS Code offers robust integration with GitHub:

*   **Source Control View**: Manage Git operations (stage, commit, push, pull) directly from the Source Control panel.
*   **GitHub Extensions**: Use extensions like "GitHub Pull Requests and Issues" to create issues, review PRs, and interact with your repository without leaving the IDE.
*   **Documentation**: Refer to the official VS Code documentation for more details: [https://code.visualstudio.com/docs/sourcecontrol/github](https://code.visualstudio.com/docs/sourcecontrol/github)
